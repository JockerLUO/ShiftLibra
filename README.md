## ShiftLibra介绍

模仿对象
Elk - 旅行货币转换器 -- Apple Design Awards 获奖作品

原作地址 : https://itunes.apple.com/us/app/elk-travel-currency-converter/id1189748820?mt=8

####  目前Pod管理的第三方插件
```
    pod 'SnapKit', '~> 3.0.0'

    pod 'AFNetworking'

    pod 'YYModel'

    pod 'pop'

    pod 'FMDB'

    pod 'FoldingCell', '~> 2.0.3'

    pod 'TPKeyboardAvoiding'
```
